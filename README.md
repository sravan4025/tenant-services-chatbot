# Tenant Services Chatbot

## Description
An NLP-based Tenant Services Chatbot that provides instant responses to tenant queries using compressed lease agreements and maintenance procedures.

## Features
- Rent and lease information
- Maintenance issue handling
- NLP-based semantic matching
- Faster tenant service resolution

## Technologies Used
- Python
- NLTK
- Scikit-learn
- TF-IDF
- Google Colab

## How to Run
1. Open the notebook in Google Colab
2. Run all cells
3. Ask tenant-related queries

## Future Enhancements
- LLM integration
- Web-based chatbot
- PDF lease ingestion
